PTA打印沙漏解题思路

Created2022-11-05|Updated2022-11-05|C语言
|Word count:638|Reading time:2min|Post View:|Comments:

难点

1.需要计算出给定数值能够打印出多少行字符。

2.需要找到每行字符长度与行数的关系。

3.对于循环嵌套的理解。

解决思路

通过找规律不难发现，总行数的一半与给定值的关系可以是：sqrt(2(n+1)/2).*

于是，我们算出了一半有多少行，将整个沙漏分为两个部分，字符数递减的和字符数递增的，然后我们就可以分为两个循环来解决。

我们也能找出行数与字符数 之间的关系，即每行字符数=2*行数+1。

代码实现

首先，我们要计算出一半有多少行

int n,c;
scanf("%d",&n);
c = sqrt(2*(n+1)/2);  //得出结果  这里的c加上只有一个字符的那一行

然后，写出上下两部分的循环

for(i=1;i<=c;i++)  //输出多少行
{
    for(m=i-1;m>0;m--)  //输出空格数，随着行数增加，空格数减少。
    {
        printf(" ");
    }
    for(k=0;k<=2*(n-i)+1;k++)  //输出字符。
    {
        printf("%c",ch);     //需要注意的是，这里将中间那一个字符也输出了的。
                             //所以下半行就不用从中间那个字符开始
    }
    printf("\n");  //字符输出完后，回车换行。
}
for(i=1;i<c;i++)  
{
    for(m=1;m<n-1;m++) 
    {
        printf(" ");
    }
    for(k=1;k<=2*i+1;i++)  //简单易懂
    {
        printf("%c",ch);
    }
    printf("\n");
}

以上，就是算法片段。

总结

这道题的解决关键就是要弄清楚到底给定值能输出几行，然后就是要找出行于数之间的关系。

完整代码

#include<stdio.h>
#include<math.h>
int main()
{
	int N,i,j,m,n;
	char ch;
	scanf("%d %c",&N,&ch);
	n=sqrt((N+1)/2); //算出一半有多少行 
	for(i=1;i<=n;i++)
	{
		for(m=i-1;m>0;m--) //随着行数增加，空格数由0，1，2，3这样依次递增，打印出所有的空格，m--，即可打印出m个空格。 
			printf(" ");
		for(j=0;j<2*(n-i)+1;j++) //随着行数增加，每行输出的字符数为2（n-i）+1，依次减少。 
			printf("%c",ch);
		printf("\n");
	}
	for(i=1;i<n;i++)    		//开始输出依次递增的行 
	{
		for(m=1;m<n-i;m++)		//与依次递减的相反，空格数依次增加。 
			printf(" ");
		for(j=1;j<=2*i+1;j++)		
			printf("%c",ch);
		printf("\n");
	}
	printf("%d\n",N-2*n*n+1);	//输出剩下的数值。 
	printf("%d",n); 
	return 0;
}
