题目

思路

本题需要将相同的字符去掉，在输出。很快就能想到是需要使用if语句来判断两组字符串中相同的字符，但难点就是判断之后该输出什么或者说要做什么处理。

注意：是从输入的第一组字符串中找到与第二组字符串相同的字符，并删除。

处理

char ch[1000000],ce[1000000];
gets(ch);
gets(ce);
for(i=0;i<strlen(ce);i++)
{
    for(j=0;j<strlen(ch);j++)
    {
        if(b[i]==a[j])
        {
            //这里面应该写什么？？？
        }
    }
}

通过理解这两个for循环，当i=0时，内循环中j从0一直到a字符串的最大值，一一检索，判断a中有没有与b相符的字符，以此类推，所以我是这样想的，能不能给一个信号，当检索到相同字符时，信号改变，于是我就写成了这样。

char ch[1000000],ce[1000000];
gets(ch);
gets(ce);
for(i=0;i<strlen(ce);i++)
{
    for(j=0;j<strlen(ch);j++)
    {
        if(b[i]==a[j])
        {
            a[j]='~';  //这里赋值符号后面可以任意写。
        }
    }
}
for(i=0;i<strlen(a);i++)
{
    if(a[i]=='~') continue;  //当判断为真时，使用continue直接执行else语句。
    else
    {
        printf("%c",a[i]);
    }
}

在PTA中成功运行。

总结

这道题主要就是要想到信号的设置，不算很难。
