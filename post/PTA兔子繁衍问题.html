PTA兔子繁衍问题

Created2022-11-06|Updated2022-11-12|C语言
|Word count:271|Reading time:1min|Post View:|Comments:

问题分析

首先需要确定兔子对数于月份的关系，通过列出对数可以发现，于斐波那契数列相一致，于是就可以把问题简单的看成时斐波那契数列于1，2，3等月份的关系。

需要特别注意：当只有一对的时候，月份为一，因为在斐波那契数列里前两个都是1。

代码编写

首先，要判断给定对数是否为一，如果为1，则输出1。

if(n==1)
{
    printf("1");
}

接下来就是利用for循环来寻找月份，具体如下：

for(i=2;next<n;i++)
{
    next=a+b;
    a=b;
    b=next;
}
printf("%d",i);

这样，就可以正确输出月份了。

需要注意的是，不可以写成next<=n；因为如果等于n时，i大于正确值。

完整代码

#include<stdio.h>
int main()
{
    int a,i,b,n,next=0,s=0;
    a=b=1;
    scanf("%d",&n);
    if(n==1)
    {
        printf("1");
    }
    else
    {
        for(i=2;next<n;i++)  
        {
        next = a+b;
        a=b;
        b=next;
        }
        printf("%d",i);
    }
    
}
