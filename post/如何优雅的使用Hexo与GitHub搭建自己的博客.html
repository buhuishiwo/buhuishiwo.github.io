如何优雅的使用Hexo与Github搭建自己的博客

准备阶段

  • 一台windows系统电脑
  • Hexo与Github环境
  • 获得个人域名（非必要）

开始

  1. 在GitHub上创建个人仓库

进入GitHub官网之后登录Github账号（如果没有自行注册）

登录之后点击Create a new repository

进入这个界面

在Repository name中输入用户名+github.io以创建仓库域名。之后再点击底部Create repository即可

以上，github个人仓库创建完成

​ 2.安装Hexo及其环境

首先，需要安装git，其官网为https://git-scm.com/，之后点击Download for windows

跳转页面之后再点击Click here to download

安装时全部点击next即可，无需其他操作，待安装完成之后运行命令行检查是否安装完成

按住win键+R键再输入cmd即可打开命令行，在其中输入

git -v

出现以下内容则安装成功，若没有出现则需重新安装，具体教程可在网上搜索（后期我也会出教程）

安装Git之后，需要将Git与你的Github仓库绑定。

具体操作如下：

鼠标右键打开Git Bash

在Git Bash中输入以下代码

git config --global user.name "你的Github用户名"
git config --global user.email "你的Github注册邮箱"

生成ssh密钥文件

ssh-keygen -t rsa -C "你的Github注册邮箱"

之后会显示输入框，直接按空格就行（三次），之后，在C盘中找到生成的.ssh文件夹中的id_rsa.pub密钥

注意看文件地址

双击使用记事本打开，并使用Ctrl+A全部选中并复制。

打开Github->Settings->SSH and GPG keys

点击New SSH Key

Title框中随便填就行

将之前复制的内容粘贴到Key框中

以上步骤完成之后，再次打开Git Bash输入以下代码测试绑定是否成功

ssh git@github.com

出现以上内容就安装成功了。

之后，我们需要安装Node.js 官网是https://nodejs.org/en/

点击左边绿色图标进行安装，同样的也是一直下一步就行。

安装完成之后，我们需要使用命令行检查是否安装成功，在命令行中输入

node -v
npm -v

出现版本信息则安装成功。

以上，Hexo的环境就安装完成了，接下来安装Hexo本体。

首先，我们需要创建一个文件夹，用来存放Hexo框架和之后的博客内容，可以命名为Blog，创建完成之后，在此文件夹中按住shift键，右键鼠标点击命令行

使用npm命令安装Hexo，输入：

npm install -g hexo-cli

等待安装完成（时间可能较长），安装完成之后，初始化我们的博客，右键鼠标打开Git Bash输入

hexo init blog

完成这一步之后，会创建一个blog文件夹，双击进入之后右键鼠标打开Git Bash输入以下命令：

hexo new test "创建名为test的博客"
hexo g "生成博客"
hexo s "启动预览服务"

之后会出现一个地址

在浏览器中打开这个地址就能看到我们的博客了

以上操作完成之后，我们已经可以在本地浏览我们的博客了，但是还不能通过网址访问我们的博客，因此我们还要将博客上传到我们的GitHub库中，通过访问库来访问我们的网站。

首先，打开位于blog文件夹下的 _config.yml文件，又称站点配置文件，另外，在themes文件下也有一个 _config.yml文件，又称主题配置文件，需要加以区分！

打开站点配置文件之后，在最后一行添加上以下代码：

deploy:
	type: git
	repo: Github仓库的完整域名并加上.git
	branch: main

注：需要在冒号：之后加上空格之后在输入代码
