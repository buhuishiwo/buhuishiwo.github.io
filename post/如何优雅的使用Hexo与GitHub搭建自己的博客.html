еҰӮдҪ•дјҳйӣ…зҡ„дҪҝз”ЁHexoдёҺGithubжҗӯе»әиҮӘе·ұзҡ„еҚҡе®ў

еҮҶеӨҮйҳ¶ж®ө

  • дёҖеҸ°windowsзі»з»ҹз”өи„‘
  • HexoдёҺGithubзҺҜеўғ
  • иҺ·еҫ—дёӘдәәеҹҹеҗҚпјҲйқһеҝ…иҰҒпјү

ејҖе§Ӣ

  1. еңЁGitHubдёҠеҲӣе»әдёӘдәәд»“еә“

иҝӣе…ҘGitHubе®ҳзҪ‘д№ӢеҗҺзҷ»еҪ•GithubиҙҰеҸ·пјҲеҰӮжһңжІЎжңүиҮӘиЎҢжіЁеҶҢпјү

зҷ»еҪ•д№ӢеҗҺзӮ№еҮ»Create a new repository

иҝӣе…ҘиҝҷдёӘз•Ңйқў

еңЁRepository nameдёӯиҫ“е…Ҙз”ЁжҲ·еҗҚ+github.ioд»ҘеҲӣе»әд»“еә“еҹҹеҗҚгҖӮд№ӢеҗҺеҶҚзӮ№еҮ»еә•йғЁCreate repositoryеҚіеҸҜ

д»ҘдёҠпјҢgithubдёӘдәәд»“еә“еҲӣе»әе®ҢжҲҗ

вҖӢ 2.е®үиЈ…HexoеҸҠе…¶зҺҜеўғ

йҰ–е…ҲпјҢйңҖиҰҒе®үиЈ…gitпјҢе…¶е®ҳзҪ‘дёәhttps://git-scm.com/пјҢд№ӢеҗҺзӮ№еҮ»Download for windows

и·іиҪ¬йЎөйқўд№ӢеҗҺеҶҚзӮ№еҮ»Click here to download

е®үиЈ…ж—¶е…ЁйғЁзӮ№еҮ»nextеҚіеҸҜпјҢж— йңҖе…¶д»–ж“ҚдҪңпјҢеҫ…е®үиЈ…е®ҢжҲҗд№ӢеҗҺиҝҗиЎҢе‘Ҫд»ӨиЎҢжЈҖжҹҘжҳҜеҗҰе®үиЈ…е®ҢжҲҗ

жҢүдҪҸwinй”®+Rй”®еҶҚиҫ“е…ҘcmdеҚіеҸҜжү“ејҖе‘Ҫд»ӨиЎҢпјҢеңЁе…¶дёӯиҫ“е…Ҙ

git -v

еҮәзҺ°д»ҘдёӢеҶ…е®№еҲҷе®үиЈ…жҲҗеҠҹпјҢиӢҘжІЎжңүеҮәзҺ°еҲҷйңҖйҮҚж–°е®үиЈ…пјҢе…·дҪ“ж•ҷзЁӢеҸҜеңЁзҪ‘дёҠжҗңзҙўпјҲеҗҺжңҹжҲ‘д№ҹдјҡеҮәж•ҷзЁӢпјү

е®үиЈ…Gitд№ӢеҗҺпјҢйңҖиҰҒе°ҶGitдёҺдҪ зҡ„Githubд»“еә“з»‘е®ҡгҖӮ

е…·дҪ“ж“ҚдҪңеҰӮдёӢпјҡ

йј ж ҮеҸій”®жү“ејҖGit Bash

еңЁGit Bashдёӯиҫ“е…Ҙд»ҘдёӢд»Јз Ғ

git config --global user.name "дҪ зҡ„Githubз”ЁжҲ·еҗҚ"
git config --global user.email "дҪ зҡ„GithubжіЁеҶҢйӮ®з®ұ"

з”ҹжҲҗsshеҜҶй’Ҙж–Үд»¶

ssh-keygen -t rsa -C "дҪ зҡ„GithubжіЁеҶҢйӮ®з®ұ"

д№ӢеҗҺдјҡжҳҫзӨәиҫ“е…ҘжЎҶпјҢзӣҙжҺҘжҢүз©әж је°ұиЎҢпјҲдёүж¬ЎпјүпјҢд№ӢеҗҺпјҢеңЁCзӣҳдёӯжүҫеҲ°з”ҹжҲҗзҡ„.sshж–Үд»¶еӨ№дёӯзҡ„id_rsa.pubеҜҶй’Ҙ

жіЁж„ҸзңӢж–Үд»¶ең°еқҖ

еҸҢеҮ»дҪҝз”Ёи®°дәӢжң¬жү“ејҖпјҢе№¶дҪҝз”ЁCtrl+Aе…ЁйғЁйҖүдёӯе№¶еӨҚеҲ¶гҖӮ

жү“ејҖGithub->Settings->SSH and GPG keys

зӮ№еҮ»New SSH Key

TitleжЎҶдёӯйҡҸдҫҝеЎ«е°ұиЎҢ

е°Ҷд№ӢеүҚеӨҚеҲ¶зҡ„еҶ…е®№зІҳиҙҙеҲ°KeyжЎҶдёӯ

д»ҘдёҠжӯҘйӘӨе®ҢжҲҗд№ӢеҗҺпјҢеҶҚж¬Ўжү“ејҖGit Bashиҫ“е…Ҙд»ҘдёӢд»Јз ҒжөӢиҜ•з»‘е®ҡжҳҜеҗҰжҲҗеҠҹ

ssh git@github.com

еҮәзҺ°д»ҘдёҠеҶ…е®№е°ұе®үиЈ…жҲҗеҠҹдәҶгҖӮ

д№ӢеҗҺпјҢжҲ‘д»¬йңҖиҰҒе®үиЈ…Node.js е®ҳзҪ‘жҳҜhttps://nodejs.org/en/

зӮ№еҮ»е·Ұиҫ№з»ҝиүІеӣҫж ҮиҝӣиЎҢе®үиЈ…пјҢеҗҢж ·зҡ„д№ҹжҳҜдёҖзӣҙдёӢдёҖжӯҘе°ұиЎҢгҖӮ

е®үиЈ…е®ҢжҲҗд№ӢеҗҺпјҢжҲ‘д»¬йңҖиҰҒдҪҝз”Ёе‘Ҫд»ӨиЎҢжЈҖжҹҘжҳҜеҗҰе®үиЈ…жҲҗеҠҹпјҢеңЁе‘Ҫд»ӨиЎҢдёӯиҫ“е…Ҙ

node -v
npm -v

еҮәзҺ°зүҲжң¬дҝЎжҒҜеҲҷе®үиЈ…жҲҗеҠҹгҖӮ

д»ҘдёҠпјҢHexoзҡ„зҺҜеўғе°ұе®үиЈ…е®ҢжҲҗдәҶпјҢжҺҘдёӢжқҘе®үиЈ…Hexoжң¬дҪ“гҖӮ

йҰ–е…ҲпјҢжҲ‘д»¬йңҖиҰҒеҲӣе»әдёҖдёӘж–Үд»¶еӨ№пјҢз”ЁжқҘеӯҳж”ҫHexoжЎҶжһ¶е’Ңд№ӢеҗҺзҡ„еҚҡе®ўеҶ…е®№пјҢеҸҜд»Ҙе‘ҪеҗҚдёәBlogпјҢеҲӣе»әе®ҢжҲҗд№ӢеҗҺпјҢеңЁжӯӨж–Үд»¶еӨ№дёӯжҢүдҪҸshiftй”®пјҢеҸій”®йј ж ҮзӮ№еҮ»е‘Ҫд»ӨиЎҢ

дҪҝз”Ёnpmе‘Ҫд»Өе®үиЈ…HexoпјҢиҫ“е…Ҙпјҡ

npm install -g hexo-cli

зӯүеҫ…е®үиЈ…е®ҢжҲҗпјҲж—¶й—ҙеҸҜиғҪиҫғй•ҝпјүпјҢе®үиЈ…е®ҢжҲҗд№ӢеҗҺпјҢеҲқе§ӢеҢ–жҲ‘д»¬зҡ„еҚҡе®ўпјҢеҸій”®йј ж Үжү“ејҖGit Bashиҫ“е…Ҙ

hexo init blog

е®ҢжҲҗиҝҷдёҖжӯҘд№ӢеҗҺпјҢдјҡеҲӣе»әдёҖдёӘblogж–Үд»¶еӨ№пјҢеҸҢеҮ»иҝӣе…Ҙд№ӢеҗҺеҸій”®йј ж Үжү“ејҖGit Bashиҫ“е…Ҙд»ҘдёӢе‘Ҫд»Өпјҡ

hexo new test "еҲӣе»әеҗҚдёәtestзҡ„еҚҡе®ў"
hexo g "з”ҹжҲҗеҚҡе®ў"
hexo s "еҗҜеҠЁйў„и§ҲжңҚеҠЎ"

д№ӢеҗҺдјҡеҮәзҺ°дёҖдёӘең°еқҖ

еңЁжөҸи§ҲеҷЁдёӯжү“ејҖиҝҷдёӘең°еқҖе°ұиғҪзңӢеҲ°жҲ‘д»¬зҡ„еҚҡе®ўдәҶ

д»ҘдёҠж“ҚдҪңе®ҢжҲҗд№ӢеҗҺпјҢжҲ‘д»¬е·Із»ҸеҸҜд»ҘеңЁжң¬ең°жөҸи§ҲжҲ‘д»¬зҡ„еҚҡе®ўдәҶпјҢдҪҶжҳҜиҝҳдёҚиғҪйҖҡиҝҮзҪ‘еқҖи®ҝй—®жҲ‘д»¬зҡ„еҚҡе®ўпјҢеӣ жӯӨжҲ‘д»¬иҝҳиҰҒе°ҶеҚҡе®ўдёҠдј еҲ°жҲ‘д»¬зҡ„GitHubеә“дёӯпјҢйҖҡиҝҮи®ҝй—®еә“жқҘи®ҝй—®жҲ‘д»¬зҡ„зҪ‘з«ҷгҖӮ

йҰ–е…ҲпјҢжү“ејҖдҪҚдәҺblogж–Үд»¶еӨ№дёӢзҡ„ _config.ymlж–Үд»¶пјҢеҸҲз§°з«ҷзӮ№й…ҚзҪ®ж–Үд»¶пјҢеҸҰеӨ–пјҢеңЁthemesж–Үд»¶дёӢд№ҹжңүдёҖдёӘ _config.ymlж–Үд»¶пјҢеҸҲз§°дё»йўҳй…ҚзҪ®ж–Үд»¶пјҢйңҖиҰҒеҠ д»ҘеҢәеҲҶпјҒ

жү“ејҖз«ҷзӮ№й…ҚзҪ®ж–Үд»¶д№ӢеҗҺпјҢеңЁжңҖеҗҺдёҖиЎҢж·»еҠ дёҠд»ҘдёӢд»Јз Ғпјҡ

deploy:
	type: git
	repo: Githubд»“еә“зҡ„е®Ңж•ҙеҹҹеҗҚе№¶еҠ дёҠ.git
	branch: main

жіЁпјҡйңҖиҰҒеңЁеҶ’еҸ·пјҡд№ӢеҗҺеҠ дёҠз©әж јд№ӢеҗҺеңЁиҫ“е…Ҙд»Јз Ғ

дҫӢеҰӮдёӢеӣҫпјҡ

д»ҘдёҠж“ҚдҪңе®ҢжҲҗд№ӢеҗҺе°ұеҸҜд»ҘйҖҡиҝҮиҫ“е…ҘжҲ‘д»¬GitHubд»“еә“зҡ„еҹҹеҗҚжқҘи®ҝй—®жҲ‘д»¬зҡ„еҚҡе®ў

дҫӢеҰӮ

д»ҘдёҠпјҢж•ҷзЁӢз»“жқҹ
